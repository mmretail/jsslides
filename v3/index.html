JS Slides v3

Where semantic HTML meets interactive slide deck

Created by @0ndras. Hit Space or → to continue…

Semantic HTML5

  • This page is a normal HTML document
  • Interactivity added by one extra script
  • Accessible, configurable, extensible

Features

  • Themeable interface
  • Navigate slides and subsections by keyboard or touches
  • Adjust behavior as needed by extension modules

Modules

  • Many modules included by default
  • Can be configured or disabled
  • Writing own modules is easy (but not necessary)

Modules: core

  • Help module: hit ? to see help
  • Overview module: hit O to toggle overview mode
  • Keyboard module: navigate slides using intuitive controls

Modules: visual

  • Font size: adjust font size using B, S, N
  • Skin: define any visual property using standard CSS
  • Transition: pick from one of the available transition animations
  • Syntax: apply syntax highlighting
    while (var i = 100) { i--; } // loop

Modules: miscellaneous #1

  • Title: adjust window title to match current slide
  • URL: Leverage browser history and slide permalinks
  • Language: switch available languages using the L key

Modules: miscellaneous #2

  • Progress: display progress meter
  • Time: display remaining time
  • Touch: use tapping, swiping and pinching to emulate keyboard on touch devices
  • Presenter's window: hit P to display comments and other useful metadata in a separate window
